# Homebrew Tap: commure/homebrew-packages

This is a **public Homebrew tap** for distributing custom formulae and casks.

## Usage

1. **Add the tap:**
    ```sh
    brew tap commure/packages
    ```

2. **Install a formula:**
    ```sh
    brew install commure/packages/python@3.9.19
    ```

3. **Upgrade formulae:**
    ```sh
    brew update
    brew upgrade
    ```
